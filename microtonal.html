Microtonal Works

For Six Strings in Fixed Tuning

The instrumentation for this piece is two violas, two cellos, and two basses. Below is a list of all the possible retunings for each open string. The distribution of these is to be determined by the group; however, all 12 pitch classes must be accounted for.

Each player will only have a portion of the 12 pitch classes. Below is a sequence to aid in group tuning. *A and Ab are +/- 0 from equal temperament. The pitches are arranged to be tuned left to right. Each measure represents a series of notes which are in tune with each other. Black notes have already be tuned.

Performance Instructions

Determine a total length.

Each box represents a string or string combination.

Move between each box in any order, repeating as often as necessary.

Notes should be held, but the duration is up to the player.

The player can take as much or as little time between entrances as desired.

When time is up, stop playing.

Performance Instructions:

Play each cell as fast as possible. If it is difficult to play the cell fast, play at eighth note = 100bpm.

The cells can be performed in the order they are arranged, however it is fine to move through them in any order and for any duration.

The page can be repeated identically or in different ways to achieve the desired length of the piece.

Cells can be octave transposed within the range of C2-E5, as octaves or as tenths.

If a cell is difficult to play, it can be practiced in a performative way by cycling through the various rhythms which are used to practice.

A small fermata can be placed on the last note of the cell. In this case, the cell should be played as a flourish.

Tuning instructions:

Tune from C2-E4

Mute every other string so that when you play octaves one note plays one string and the other plays multiple.

Tune the strings while playing the octave to achieve a microtonal pitch collection.

Each octave should have at least three pitches sounding.
