Microtonal Scores

For Six Strings in Fixed Tuning

The instrumentation for this piece is two violas, two cellos, and two basses.

Below is a list of all the possible retunings for each open string.

The distribution of these is to be determined by the group;

however, all 12 pitch classes must be accounted for.

Each player will only have a portion of the 12 pitch classes.

Below is a sequence to aid in group tuning.

*A and Ab are +/- 0 from equal temperament.

The pitches are arranged to be tuned left to right.

Each measure represents a series of notes which are in tune with each other.

Black notes have already be tuned.

Performance Instructions

Determine a total length based on what is physically comfortable.

Each box represents a string or string combination.

1

Cycle through the different string combinations in each box

in any order, random or premeditated,

as many times as the determined length permits.

The length of each box and rate of change is also up to the player,

however the bowing should be constant for the entire piece.

Hanon as Materials

Tuning instructions:

Strip mute the following notes so that only one string is left unmuted:

D2, F2, A2, C3, E3, G3.

This will result in two or more strings of the following notes being unmuted.

C2, E2, G2, D3, F3, A3.

This will result in octaves which produce three or more pitches, which are represented as vertical pairs.

[Horizontally] the top row is a list of keys which produce one pitch and the bottom row is a list of keys which produce two or more.

C3, E3, G3, D2, F2, A2

C2, E2, G2, D3, F3, A3

Tune each vertical row.

The tuning should use a mix of just and beating intervals to achieve a pitch cloud for each vertical row.

Performance Instructions:

Play the first five octaves of the first measure of The Virtuoso Pianist by Charles-Louis Hanon as fast as possible.

The fifth octave should be held for a duration equal to the total length of the first four octaves.

Repeat.

Ravel and Tenney as Materials

For Quarter Tone Pianos

Dots are arranged as four phrases and should be read from left to right.

The first phrase is at the top and the last at the bottom.

The dots represent the general density of the two pianos and should not be scrutinized.

The pianos use the following three chords as material,

played and repeated in a randomized order to achieve the density represented in each phrase.

Use a stopwatch and start the first phrase at 00:00.

Move through the first phrase and reach the end at 0:45.

Pause.

At 0:50, the higher piano plays the following excerpt as written.

Start the next phrase at 1:00.

Continue this pattern until the stopwatch hits 4:00.

The higher piano plays the aforementioned excerpt at the ends of phrases 1 and 3,

and the lower piano plays the same excerpt at the ends of phrases 2 and 4.
