Microtonal Works

For Six Strings in Fixed Tuning

The instrumentation for this piece is two violas, two cellos, and two basses.

Below is a list of all the possible retunings for each open string.

The distribution of these is to be determined by the group;

however, all 12 pitch classes must be accounted for.

Each player will only have a portion of the 12 pitch classes.

Below is a sequence to aid in group tuning.

*A and Ab are +/- 0 from equal temperament.

The pitches are arranged to be tuned left to right.

Each measure represents a series of notes which are in tune with each other.

Black notes have already be tuned.

Performance Instructions

Determine a total length.

Each box represents a string or string combination.

Move between each box in any order, repeating as often as necessary.

Notes should be held, but the duration is up to the player.

The player can take as much or as little time between entrances as desired.

When time is up, stop playing.

Tuning instructions:

Tune from C2-E5

Strip mute every other white key so that when you play octaves one note plays one string and the other plays multiple.

Tune the strings while playing the octave to achieve a microtonal pitch collection.

Each octave should have at least three pitches sounding.

Performance Instructions:

Play each cell as fast as possible. If it is difficult to play the cell fast, play at eighth note = 100bpm.

The page can be repeated as written or in different ways to achieve the desired length of the piece.

Each cell can be played in octaves or with the right hand a M10 above the left.

This can be done in any octave within the range of C2-E5.

If a cell is difficult to play, it can be practiced while performing by cycling through the various rhythms which are used to practice.

The last pitch of the cell can be held for a duration equal to the total length of the rest of the notes in the cell.
